# Banking AML Streaming Pipeline

Enterprise-style Anti Money Laundering (AML) streaming project using:

- Python Faker
- Confluent Kafka
- Databricks Community Edition
- PySpark Structured Streaming
- Delta Lake
- Medallion Architecture

## Project Goals

- Generate realistic banking datasets
- Stream data into Kafka
- Ingest into Databricks Bronze layer
- Transform into Silver/Gold layers
- Detect suspicious AML activities
- Mimic enterprise deployment standards
